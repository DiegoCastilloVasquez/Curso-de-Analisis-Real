# Curso de Análisis Real I

Material didáctico para el curso de **Análisis Real I**, elaborado por el **Centro de Estudiantes de Matemática (CEM)** de la Facultad de Ciencias Matemáticas (FCM) de la **Universidad Nacional Mayor de San Marcos (UNMSM)**.

## 📖 Acerca de

Este proyecto surge ante la necesidad concreta de material académico disponible en idioma español para el curso de Análisis Real I, correspondiente al período lectivo 2026. El CEM, como organización representativa del alumnado, ha tomado la iniciativa de redactar el presente material con el objetivo de ser de auténtico provecho al estudiantado en su proceso de aprendizaje y aprobación de la materia.

## 📋 Contenido

El documento cubre los siguientes temas fundamentales:

   - Conjuntos y operaciones entre conjuntos
   - Funciones (inyectivas, sobreyectivas, biyectivas)
   - Composición de funciones
   - Familias de conjuntos
   - Producto cartesiano
   - Números naturales y axiomas de Peano
   - Buena ordenación y principio de inducción
   - Conjuntos finitos e infinitos
   - Conjuntos enumerables
   - Conjuntos no enumerables y argumento diagonal de Cantor

## 🛠️ Compilación

Para compilar el documento LaTeX, asegúrate de tener una distribución de LaTeX instalada (TeX Live, MiKTeX, etc.) y ejecuta:

```bash
pdflatex main.tex
```

Si es necesario, ejecútalo dos o tres veces para resolver las referencias y el índice:

```bash
pdflatex main.tex
pdflatex main.tex
```
